# Hello Machi FM — GitHub Pages Portal

Hello Machi FM live radio portal — hosted on GitHub Pages.

## Hosted At
https://fmhellomachi.github.io/hello-machi-portal/

## Files
- `index.html` — The portal web app
- `chat.html` — Standalone mobile chat page
- `config.json` — Station config (read-only static fallback)
- `logo.jpg` — Station logo

## Editing Station Details (Admin)
1. Open the portal URL
2. Tap the ⚙️ Settings icon
3. Login with your admin credentials
4. Update station settings and click **SAVE & SYNC**
   - Changes are saved to the **Render backend** config
   - All listeners and the Android app update automatically

## Backend
Chat and config API is hosted on Render:
https://hello-machi-backend.onrender.com
